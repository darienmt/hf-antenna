## Development

### Create venv and install dependencies

```shell
python3 -m venv .venv
. ./.venv/bin/activate
python3 -m pip install -r ./requirements.txt
```

### Update Dependencies

```shell
python3 -m venv .venv
. ./.venv/bin/activate
python3 -m pip freeze > ./requirements.txt

```
